About JustWeather

JustWeather is a lightweight weather application designed to deliver accurate, reliable, and flexible weather information and forecasts. It was developed as a personal project by Vojin Mitrović.

Features

  • Real-Time Weather Data: Integrates with the OpenWeather API to provide accurate and up-to-date weather information for any selected location.
  • Automatic Location Detection: Enables weather updates for the user’s current location, based on geolocation permissions.
  • Custom Location Search: Utilizes the HERE API to allow manual entry of locations for tailored weather forecasts.

Purpose

JustWeather was created to demonstrate web development expertise and explore the integration of AI in coding workflows. While AI tools were used during development, all functionalities have been manually tested and refined to ensure quality and reliability.

Documentation

The development of the JustWeather app is guided by comprehensive product documentation, including the Product Requirements Document (PRD), Software Requirements Specification (SRS), and a Sprint Plan with a detailed task sheet.

The app has undergone two cycles of testing: first in the staging environment and then in the production environment. The testing process is recorded in a Traceability Matrix and Test Cases Sheet to ensure thorough coverage and accountability. Two bugs were identified during the initial test run, both of which were resolved and verified in the subsequent production environment test run.
